== Description ==

Using Post export plugin admin can bulk export posts from the bulk option dropdown present on post listing page, also admin can select which attribute to export via checkboxes.

== Installation ==

1. Unzip/Unrar the plugin folder and copy all the files to YOUR_SITE/wp-content/plugins/ folder.
2. You can also use wordpress plugin uploader to do so.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Make sure to give Upload folder writing permission as this plugin will create a file to download there.

== Frequently Asked Questions ==

= What exactly this plugin do ?

This plugin is used to export all posts.
